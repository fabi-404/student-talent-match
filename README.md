# student-talent-match
Der Student Talent Matcher ist eine Web Anwendung, die Studierende und Absolventen direkt mit einstiegsnahen IT-Positionen (Praktika, Werkstudenten, Junior-Rollen) im deutschen Markt zusammenbringt. Die App bietet eine klare, datengestützte Analyse der Übereinstimmung zwischen akademischen Fähigkeiten/Projekten und Jobanforderungen.

## 1. Outline der Value Proposition

Der **Student Talent Matcher** ist eine Web Anwendung, die Studierende und Absolventen direkt mit **einstiegsnahen IT-Positionen** (Praktika, Werkstudenten, Junior-Rollen) zusammenbringt. Die Web-App bietet eine klare, datengestützte Analyse der Übereinstimmung zwischen akademischen Fähigkeiten und Jobanforderungen.

### Problem

- **Die Herausforderung:** Studierende wissen oft nicht, welche ihrer akademischen Skills (Projekte, Vorlesungen) in der Praxis relevant sind. Das Suchen auf generischen Plattformen ist ineffizient.
- **Die Ineffizienz:** Studierende bewerben sich ohne klare Passgenauigkeit; Unternehmen übersehen qualifizierte Junior-Talente.

### Lösung (Die App)

Die App fungiert als **"Student Talent Matcher"** .

1. **Fähigkeiten-Erfassung:** Nutzer geben ihre Skills und relevanten **Uni-Projekte** (z.B. "Android Studio", "Datenbanken (SQL)") in Form von **Tags** ein.
2. **Einfaches Matching:** Die App vergleicht diese Tags mit den **Anforderungs-Tags** der hinterlegten **Junior-Job-Profile** und schlägt passende  Stellenausschreibungen vor.
3. **Fokus auf Lücken:** Die App zeigt Studierenden klar, welche **Lücken** sie für den Traumjob schließen müssen.

### Target User(s)

1. **Studierende / Absolventen (Hauptzielgruppe):** Informatik, Wirtschaftsinformatik und verwandte Studiengänge, die Praktika, Werkstudentenstellen oder den ersten Job suchen.
2. **Administrator/Content Manager (Sekundär):** Interne Rolle zur Pflege und Eingabe neuer, auf **Einstiegslevel** zugeschnittener Job-Profile.
