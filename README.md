# student-talent-match
Der Student Talent Matcher ist eine mobile Anwendung, die Studierende und Absolventen direkt mit einstiegsnahen IT-Positionen (Praktika, Werkstudenten, Junior-Rollen) im deutschen Markt zusammenbringt. Die App bietet eine klare, datengestützte Analyse der Übereinstimmung zwischen akademischen Fähigkeiten/Projekten und Jobanforderungen.
